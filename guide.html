For those who are unaware, machine learning is a GPU intensive process that requires high-end hardware to run efficiently, i.e gaming desktops. To combat this issue and create an accessible library that is available to everyone, it is recommended to use the Google Colab development platform.

Dataset

Within all machine learning models, a good initial data set to train the model is essential. All datasets must contain labelled data in order to teach the model what the object is and must also be split into a training and test set (aprox 4:1 split ), this enables the model to assess it’s accuracy by viewing data it has not yet seen, however, for time series data it is always best to use the earliest data to train the model and the latest data to test it in order to identify it’s effectiveness at predicting future results.

Random Forests

Random forests are several single comparison models that gather an aggregated mean of all the outcomes to produce a reliable output. The method for defining a random forest is:

1. Randomly choose a subset of the rows of your data.

2. Train a model using this subset.

3. Save that model, and then return to step 1 a few times.

4. To make a prediction, predict using all the models, and then take the average of each of those model's predictions.

Unlike standard decision trees, this method contains zero-averaging error as there are no dependencies that could potentially skew the output data, instead the model uses the results of a series of independent comparisons to create an accurate prediction
